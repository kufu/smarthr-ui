# CheckBoxLabel

```tsx
import { CheckBoxLabel } from 'smarthr-ui'

;<CheckBoxLabel
  label="CheckBox label"
  name="sample"
  checked={true}
  mixed={true}
  onChange={() => {}}
  themeColor="light"
/>
```

## props

CheckBoxLabel props extends CheckBox props.

In addition to it...

| Name  | Required | Type       | DefaultValue | Description                        |
| ----- | -------- | ---------- | ------------ | ---------------------------------- |
| label | true     | **string** | -            | Text to the right of the CheckBox. |
